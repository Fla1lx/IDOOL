# IDOOL

IDOOL is a mobile idle / tycoon game about growing a small auto repair shop into a busy workshop.

## Project

- **Genre:** mobile idle / tycoon
- **Setting:** auto repair shop
- **Engine:** Unity
- **Current stage:** repository preparation

## MVP Goal

The MVP should prove the basic idle loop: a car arrives, the player starts a repair, a timer completes, the player earns money, and that money is spent on workshop upgrades.

At this stage the repository is prepared for safe Unity development. The Unity project should be opened or created through Unity Hub, without manually generating large Unity service files.
